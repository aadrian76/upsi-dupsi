## Cloud Security Fundamentals

The following course provides learners a solid starting point to understand key cloud security topics. It covers the basics, like what the cloud is, the main roles involved, and the different ways cloud services can be set up — from public and private clouds to hybrid and multi-cloud environments. Learners will also get familiar with the main characteristics that make cloud services unique, like flexibility, easy access, and shared resources. On top of that, it introduces some of the core technologies that make cloud systems work, such as virtualization, containers, identity management, and basic security tools. With this foundation, learners will be ready to dive into more advanced cloud security concepts later on.

## Service Line Cloud SecOps Catalog

Objective: Gaining Insights on the CSO Catalog Overview

  

As part of this session, you will meet with leaders from each cloud provider to gain a comprehensive understanding of the CSO Catalog. During the session, the leaders will share insights into the structure of the catalog, how it is organized, and the key processes involved in its development and management. Additionally, they will explain how each cloud provider contributes to the catalog, their ways of working, and how they align with the broader security objectives. This collaborative discussion will help provide clarity on roles, responsibilities, and best practices across the different providers, ensuring a unified approach to cloud security.
