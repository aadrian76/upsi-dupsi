# Week 3 - CSO

Welcome to Week 3 of your training journey! This week marks an important step as you begin working directly with a leading cloud provider. Over the next few days, you will start building your technical foundation, learning how to interact with cloud services and set up your own working environments.

  

We’ll kick off with an introduction to command line tools, an essential skill for cloud interactions. Whether you are managing infrastructure, automating processes, or configuring services, command line proficiency will be key. Next, you’ll move into environment setup, where you’ll create the accounts, profiles, or project spaces required to follow along with the hands-on portions of your training. Finally, the week will culminate with a foundational cloud course, which will solidify your understanding of core concepts and basic services.

  

This week is all about getting comfortable with the tools, environments, and learning paths you’ll be using throughout the rest of your cloud journey. Let’s dive in!

  

*From this point forward, you will need to follow the training materials for your assigned cloud provider.

If you are unsure which provider has been assigned to you, please contact your tutor or training lead for clarification.*

  
  

### Command line introduction

The command line is a powerful interface for interacting with cloud environments. In this section, you’ll learn how to navigate the command line, understand essential commands, and gain familiarity with cloud provider-specific command line interfaces (CLIs). Mastering the command line will help you automate tasks, deploy resources, and troubleshoot environments efficiently. Whether you’re launching virtual machines, managing storage, or configuring permissions, the command line will become a vital tool in your cloud toolkit.

Completing this course will prepare you for deeper technical training in the weeks ahead.
